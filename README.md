# Student Grade Tracker

## Description
A Java console-based application to manage student grades and calculate average, highest, and lowest scores.

## Features
- Add multiple students
- Calculate average marks
- Find highest and lowest marks
- Display complete report

## Technologies Used
- Java
- ArrayList
- OOP Concepts

## How to Run
1. Compile: javac StudentGradeTracker.java
2. Run: java StudentGradeTracker
